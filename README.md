# self-introduction-practice

##　概要
COACHTECH 教材 Tutorial 5-1「HTMLの基礎 ハンズオン演習」で作成した成果物です。
見出し<h1~h2>,リストタグ:<ul><ol><li><div> ブロックレベルのタグの使用
画像の埋め込み: <img>,リンクの飛び先を指定する:<href> (Hypertext Reference)の使用

##　使用技術
- HTML5
- GitHub

## 学んだこと
HTMLの基本構造(DOCTYPE/html/head/body)の役割
見出し（h1~h6）の使い分け,<p>段落</p>,画像やリンクの埋め込みの仕方
<label>によるアクセシビリティ,ユーザビリティ。
<button>などHTMLフォームの基礎

## 動作確認
ファイルを保存→パスをコピー→ブラウザにパスを貼り付けて開く
リストが箇条書き・番号付きで表示されているか。
リンクがクリック・新しいタブで開くか。
各フォームに入力可能か,ラベルをクリックするとフィールドにフォーカスが当たるか
必須項目を空にして送信しようとすると警告がでるか
